# Montara

A real estate tools portal that provides quick access to specialized applications.

## Live Site

https://montara-portal.web.app

## Skills

| App | Description | URL |
|-----|-------------|-----|
| **Disclosure AI** | AI-powered real estate disclosure document analysis | https://disclosureai-7362d.web.app |
| **HomeMatch** | Match buyers with sellers using intelligent algorithms | https://homematcher-86e14.web.app |

## Tech Stack

- **Frontend:** React 18 + TypeScript
- **Build:** Vite
- **Styling:** Tailwind CSS
- **Hosting:** Firebase Hosting

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

```bash
# Deploy to Firebase Hosting
npm run deploy
```
